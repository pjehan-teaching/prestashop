Développer un thème

StarterTheme

La documentation officielle recommande l’utilisation du Starter Theme pour développer un nouveau thème PrestaShop à partir d’une base solide.

cd themes
git clone https://github.com/PrestaShop/StarterTheme.git YOUR_THEME_NAME
StarterTheme

Fichier theme.yml

Commencez par créer le fichier theme.yml en copiant le fichier config/theme.dist.yml

Adaptez ensuite ce fichier en fonction de vos besoins :

name: YOUR_THEME_DIRECTORY_NAME
display_name: YOUR THEME NAME
version: 1.0.0
author:
  name: "PrestaShop Team"
  email: "pub@prestashop.com"
  url: "https://www.prestashop.com"

meta:
  compatibility:
      from: 1.7.3.0
      to: ~

Gestion des assets

Le dossier assets contient les assets compilés pour la production et le dossier _dev les fichiers bruts.

Nous allons commencer par compiler les assets en utilisant webpack :

cd _dev
npm run watch

Créer un hook

Déclarer le nouveau hook dans le fichier config/theme.yml :

global_settings:
  hooks:
    custom_hooks:
      - name: displayFooterBefore
        title: displayFooterBefore
        description: Add a widget area above the footer

Appeler le hook dans un fichier template :

{hook h='displayFooterBefore'}
