Écrire une requête SQL

Pour créer une requête SQL, appuyez sur le bouton "Ajouter une requête SQL" puis choisissez le nom que vous souhaitez donner à votre requête.

Écrivez ensuite le code SQL de votre requête.

Exemple pour récupérer la liste de l'ensemble des produits :

SELECT * FROM ps_product;
Le prefixe ps_ peut varier en fonction de votre installation !

Écrire une requête SQL

  1. Afficher la liste des produits ayant un prix compris entre 20 et 50 euros.
  2. Afficher la liste des employés actifs triés par Nom puis par Prénom.
  3. Afficher le nombre de comptes client créés au cours du dernier mois.
  4. Afficher le prix moyen des produits.
  5. Afficher la liste des points d'accroche avec le nombre de module qu'ils contiennent.
