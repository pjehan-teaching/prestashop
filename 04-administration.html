L'interface d'administration - Tableau de bord

Lorsque vous vous connectez pour la première fois à votre administration PrestaShop, vous arrivez sur le tableau de bord.

Cette page présente des statistiques du site ainsi que des conseils concernant les réglages.

L'interface d'administration - Les menus

Un menu supérieur vous affiche le logo de PrestaShop ainsi que le numéro de version.

Un menu déroulant vous permet d'afficher des liens vers des accès rapides (liste des commandes, création d'un produit...).

Une barre de recherche vous permet également de rechercher rapidement un contenu spécifique de votre site.

L'interface d'administration - Les menus

En haut à droite de l'écran, vous trouverez un lien vous permettant d'afficher votre boutique.

Des icônes vous permettent également de consulter vos notifications et un autre vous affichera vos badges.

L'interface d'administration - Les menus

  • Vendre : Permet de gérer le quotidien de votre site e-commerce (gestion des produits, des commandes, des clients...).
  • Personnaliser : Permet de modifier de gérer des fonctionnalités globales à l'ensemble de votre site e-commerce (apparences, modules, langues...).
  • Configuration : Permet de gérer des paramètres globaux du site à son démarrage (Paramètres de la boutique) et des paramètres techniques plus poussés pour par exemple optimiser le référencement (Paramètres avancés).

L'interface d'administration - Les menus

Sur la plupart des pages, un bouton Aide vous permet d'afficher une fenêtre contenant des informations sur la page en cours.

Un autre bouton peut vous permettre d'afficher les modules complémentaires pouvant être installés et en lien avec le contenu affiché.

Premières configurations - Désactiver notre site

Nous allons commencer par afficher notre site e-commerce en maintenance afin d'éviter que des internautes n'affichent le site avant qu'il soit terminé.

  • Configurer
  • Paramètres de la boutique
  • Maintenance

Activer la boutique vous permet de passer votre site e-commerce en mode maintenance.

Si vous souhaitez tout de même pouvoir accéder au site, vous pouvez ajouter votre IP dans la liste des IP de maintenance.

Il existe également un "Mode catalogue" permettant d'afficher votre site mais sans produits et sans la possibilité de passer de commande.

  • Configurer
  • Paramètres de la boutique
  • Produits

Premières configurations - Supprimer les contenus de test

Après installation, votre boutique contient des données d'exemple (produits, clients, commandes...) afin d'éviter de partir d'une page blanche.

Nous allons cependant choisir de partir de zéro et de supprimer les données existantes.

Pour cela, nous allons utiliser un module "PrestaShop Cleaner" :

  • Personnaliser
  • Modules
  • Catalogue de modules

Cliquez ensuite sur les boutons pour "Supprimer le catalogue", "Supprimer les commandes et clients", "Vérifier et réparer" et "Nettoyer et optimiser".

Contenus statiques - Gestion des menus

L'ensemble des menus du site sont gérés via des modules :

  • Menu principal : Permet de gérer le menu principal du site
  • Liste de liens : Permet de gérer les menus secondaires

Ces modules se trouvent dans la catégorie "Design & Navigation".

  1. Ajouter un lien vers l'accueil dans le menu principal
  2. Ajouter un lien vers les CGV dans le footer.

Modules - Installer un module

Vous pouvez ajouter un nouveau module en recherchant sur le marketplace :

  • Personnaliser
  • Modules
  • Catalogue de modules
  1. Installer le module "Liste des marques".

Modules - Gérer les emplacements

Les modules sont affichés dans des zones définies par le thème installé.

La liste des zones et des modules qu'elles contiennent sont affichées sur la page "Positions" :

  • Personnaliser
  • Apparence
  • Positions

Il est possible de greffer un module sur une position et de gérer l'ordonnancement des modules dans une position.

Il est également possible de greffer un même module sur plusieurs positions.

  1. Afficher le module "Liste des marques" au dessus du footer et uniquement sur la page d'accueil.
